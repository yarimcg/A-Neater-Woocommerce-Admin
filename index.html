What it does

Practical improvements that add up: faster workflows, clearer information, and less admin noise.

Quick Find

Instant search for orders, customers, products, and admin menus. Press /, type, navigate with ↑/↓, Enter to open, Esc to close.

  • Orders + customers results, “Show more”, quick status links
  • Menu search is client-side (fast, no AJAX)
  • Optional preloading for near-instant results

Order list enhancements

A more readable list table with helpful, glanceable customer context.

  • Tags: Guest / New / Returning / Wholesale
  • Cust. Orders and Cust. Revenue columns
  • Payment method column with optional icons

Customer revenue

A dedicated Customer Revenue screen: filter by timeframe, sort, search, rebuild indexes, and see first/last order at a glance.

  • Customers / Wholesalers / Guests / Company filtering
  • Purchase history + top purchased items
  • Progress tracking and logs

On Sale Manager

Manual approval workflow for the Sale product category: decide what appears on the Sale page, even when products have a sale price set.

  • Approve/exclude with checkboxes (range select)
  • Auto-sync on product + variation save
  • Re-sync All for imports/restores/first setup

Cleaner admin UI

Removes distractions and streamlines repetitive screens.

  • Hide unnecessary user profile fields
  • Mobile-friendly orders list and order edit layout
  • Reviews count surfaced in menus

Made for speed

Built around fast interactions: keyboard-first flows, lightweight UI, and admin features that stay out of the way until you need them.

Install

This is a WordPress plugin. Install it like any plugin, then open A Neater Admin in wp-admin.

Download plugin (.zip) v12.4.5
1

Upload / install

In WordPress: Plugins → Add New → Upload Plugin, then choose the downloaded zip.

2

Activate

Activate in Plugins. The plugin will create its indexes/tables as needed.

3

Use Quick Find

Press / from anywhere in wp-admin to search instantly.

Older versions

Keep versioned plugin zips in downloads/ using names like a-neater-woocommerce-admin-v12.4.5.zip.

FAQ

Does this change my storefront theme?
No. It focuses on wp-admin and WooCommerce management workflows.
What’s the fastest way to feel the difference?
Use Quick Find (/) and the enhanced Orders list columns. Those two alone save time immediately.
Where do I add screenshots?
Put images in `site/assets/screens/` (create it) and replace the placeholders in the Before/After section.
